Install

  1. Unzip the archive to a favorite folder.
  2. If you have not use English or Korean Windows, rename "en" folder to your language-code folder.
    Chinese -> zh, Deutsch -> de, French -> fr, Spanish -> es, ... (Please see this for details)
    You will be happy a little, when you extract files from GRF or you make GPF.
    (If you want to become more happy, do google about msgunfmt/msgfmt of gettext, and do effort greatly)
  3. In the addition, you may relate the extension of ACT and/or SPR.
    (e.g. c:\RO-tools\actOR2\actOR2.exe "%1")

Uninstall

  1. Delete the unzipped folder.
  2. If you relate extensions, erase relations.
This program don't use registry. it uses one ini file.

インストール

  1. 好きなフォルダにzipファイルを展開してください
  2. 追加で、ACT・SPRの関連付けをしてもいいと思います
    (例 c:\RO-tools\actOR2\actOR2.exe "%1")

アンインストール

  1. 展開したフォルダを消してください
  2. 拡張子の関連付けをしたなら、関連付けを消してください
このプログラムは、レジストリを使っていません。iniファイルを1つ使ってます。
